# TodoCo Screenshots

Drop screenshots into this folder using the filenames below.
README is already wired to these paths.

Required files:

- `tray-popup.png` (main tray popup task list)
- `markdown-editor.png` (markdown bulk edit window)
- `admin-users.png` (admin user management screen)
- `login-screen.png` (desktop login/register screen)
- `register.png` (register flow screen)
- `share.png` (realtime team sharing view)
- `workspace.png` (workspace creation/management view)

Recommended capture width:

- Desktop popup: 700-900px
- Editor/Admin: 1200-1600px
